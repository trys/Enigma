About this Enigma machine

Inspired by a visit to Bletchley Park, I decided to create a JavaScript implementation of the infamous Enigma machine used by Nazi Germany during the Second World War. It models the three rotors, reflector and lamp board providing the same encryption level as the 1928 Enigma G model.

Currently it lacks the plugboard and offset rotor turnovers but I hope to add these features shortly. That would bring the machine up to a 1939 naval-spec Enigma.

How to use this machine

To begin, set the rotor positions using the three input fields. Then choose either lamp or message mode with the toggle switch to the right of the lamp board. Then, type your message in the final box.

The only way to retrieve the encrypted message is to set the rotors to the same starting positions as when the message was typed. Any other setting and the message will be indecipherable.
